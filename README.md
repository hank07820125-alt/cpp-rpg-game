# Simple RPG Game

## GitHub Repository

https://github.com/hank07820125-alt/cpp-rpg-game

## 專案介紹

這是一個使用 C++ 製作的文字版 RPG 小遊戲。

玩家需要與怪物進行戰鬥，可以選擇普通攻擊、使用技能或使用道具，直到擊敗怪物獲得勝利。

本專題主要練習 C++ 物件導向程式設計（OOP），並實作 Player、Skill、Item 與 Monster 四個類別。

## 遊戲功能

* Normal Attack
* Use Skill
* Use Item
* HP / MP 顯示
* 勝利判定

## 使用到的 Class

* Player
* Skill
* Item
* Monster

## 編譯方式

g++ -std=c++17 main.cpp -o main

## 執行方式

./main
